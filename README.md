# SAM AI Tech Internship Tasks

This repository contains the Artificial Intelligence internship tasks completed as part of the SAM AI Technologies Internship Program.

## Completed Tasks

### Task 1 – Text Summarizer
- Summarizes long text into concise summaries.
- Built using Python and Flask.

### Task 2 – AI PDF Question Answering
- Upload a PDF and ask questions about its content.
- Uses Sentence Transformers and FAISS for semantic search.

### Task 4 – Fake News Detection
- Detects whether a news article is Fake or Real.
- Built using Machine Learning (TF-IDF + Classifier) and Flask.

## Technologies Used

- Python
- Flask
- Scikit-learn
- Sentence Transformers
- FAISS
- Pandas
- NumPy
- HTML
- CSS
